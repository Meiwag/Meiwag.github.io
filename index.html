ASTRALIS
Astralis is a solo project made by Meiwag (fnitur385) The game is hosted on ROBLOX, this game is a space exploration sandbox where every player basically plays together at the same time Players can build bases, defend places, remove faces and destroy others bases Build your own spaceship and explore the universe, conquer stars or start a faction for yourself
